- Keymap -

<D+Left>: Go to the start of the line
<D+Right>: Go to the line of the end
<D+Up>: Scroll up
<D+Down>: Scroll down
<S+Left>: Select and go left
<S+Right>: Select and go right
<S+Up>: Jump to the top
<S+Down>: Jump to the bottom

<C+f>: Focus on file explorer (Nvim-Tree)

<C+w>: Write File
<C+r>: Run File

<A+Left>: Switch to the previous tab
<A+Right>: Switch to the next tab
<A+Down>: Remove current tab

<C+t>: Open / Close terminal

- Work on [Normal], [Insert]

# Normal Mode

p: Paste text

<C+e>: Switch to Edit Mode (Insert Mode)
<C+c>: Exit

# Insert Mode

<C+c>: Switch To Normal Mode

# Require
- ascii-image-converter

# Visual Mode

y: Copy text
p: Paste text
