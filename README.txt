- Keymap -

<D+Left>: Go to the start of the line
<D+Right>: Go to the line of the end
<D+Up>: Scroll up
<D+Down>: Scroll down
<S+Left>: Select and go left
<S+Right>: Select and go right
<S+Up>: Jump to the top
<S+Down>: Jump to the bottom
<S+C+Up>: Move the current up
<S+C+Up>: Move the current down

<C+f>: Focus on file explorer (Nvim-Tree)

<C+w>: Write File
<C+r>: Run File

<A+Left>: Switch to the previous tab
<A+Right>: Switch to the next tab
<A+Down>: Remove current tab
<A+[>: Move the current tab left
<A+]>: Move the current tab right

<C-z>: Toggle lsp lines
<C+x>: Go to the next error
<C+d>: Go to definition

<C+t>: Open / Close terminal

- Works on [Normal], [Insert]

# Normal Mode

p: Paste text

<C+e>: Switch to Edit Mode (Insert Mode)
<C+c>: Exit

# Insert Mode

<C+c>: Switch To Normal Mode

# Require
- ascii-image-converter

# Visual Mode

y: Copy text
p: Paste text
a: Add stuff to the start of every selected lines
r: Remove the first character from every selected lines
